# Full-Stack-Banking-Project

## Project Description
Full-stack Banking application with a Java back-end and web-based front-end

## Technologies Used
Java 8
PostgreSQL
HTML 5
CSS 3
JavaScript
Javalin
Log4j

## Features
* Login
* Deposit/Withdraw funds
* Approve/Deny Account requests
* Persistent local external transaction log
